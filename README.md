## Starting app on a local machine

1. Clone repository:
   ```bash
   git clone https://github.com/VaultDweller13/junior-hackathon.git
   ```
2. Navigate to the project directory:

   ```bash
   cd junior-hackathon
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run one of the following commands:

   ### Dev server:

   ```bash
   npm run dev
   ```

   ### Build:

   ```bash
   npm run build
   ```

   ### Preview:

   ```bash
   npm run preview
   ```

   ### Lint:

   ```bash
   npm run lint
   ```
